¿Como funciona la virtualizacion?

La virtualización funciona mediante un software llamado hipervisor o VMM (Virtual Machine Monitor), que se instala en el sistema físico y permite crear y gestionar máquinas virtuales. El hipervisor se encarga de asignar los recursos del sistema físico a las máquinas virtuales, y de gestionar el acceso a los dispositivos de hardware. El hipervisor también se encarga de aislar las máquinas virtuales entre sí, de manera que cada máquina virtual funciona como un sistema independiente, con su propio sistema operativo y aplicaciones.
